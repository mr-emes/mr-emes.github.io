Bantuan & Panduan

Panduan untuk Guru

1. Membuat Kelas

Klik tombol "Buat Kelas" di beranda atau halaman kelas. Isi nama kelas dan deskripsi. Kode kelas akan dibuat otomatis.

2. Import Data Siswa

Buka Pengaturan → Kelola Data Siswa. Upload file CSV dengan format: NIS,Nama,Kelas atau tambah siswa satu per satu.

3. Membuat Tugas

Klik "Buat Tugas", pilih kelas, isi detail tugas dan tenggat waktu. Anda bisa menambahkan lampiran file.

4. Menilai Tugas

Di halaman tugas, klik "Lihat Pengumpulan" untuk melihat tugas yang dikumpulkan siswa dan berikan nilai.

Panduan untuk Siswa

1. Melihat Tugas

Buka halaman "Tugas" untuk melihat semua tugas dari kelas yang Anda ikuti. Perhatikan tenggat waktu!

2. Mengumpulkan Tugas

Klik "Kumpulkan Tugas" pada tugas yang ingin dikerjakan. Tambahkan catatan dan lampiran file jika diperlukan.

3. Melihat Nilai

Buka halaman "Nilai" untuk melihat nilai dan feedback dari guru untuk tugas yang sudah dinilai.

Pertanyaan Umum (FAQ)

Bagaimana cara reset password?

Hubungi administrator sekolah untuk reset password atau gunakan fitur lupa password di halaman login.

File apa saja yang bisa diupload?

Mendukung file PDF, DOC, DOCX, JPG, PNG, dan ZIP dengan maksimal ukuran 10MB per file.

Bagaimana jika terlambat mengumpulkan tugas?

Tugas masih bisa dikumpulkan setelah tenggat waktu, namun akan ditandai sebagai terlambat.
